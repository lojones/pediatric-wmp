
# Pediatric Weight Management – Care Model Explorer

Interactive single-page HTML tool designed for a multidisciplinary pediatric weight management program to evaluate **three proposed models of care** for delivering ≥25 care hours per patient over a 2-year program.

## What It Does

- **Compares 3 models** (Group-Heavy Rotation, Hybrid Tiered, Block Curriculum) across hours, visit types, group vs 1:1 split, scheduling complexity, and more
- **Visual charts** — bar charts, donut charts, and a radar comparison drawn on HTML Canvas (no external libraries)
- **Interactive tier explorer** for the tiered model (Tier 1 / 2 / 3 schedules)
- **Expandable phase cards** for the block curriculum model
- **Staffing overview** with capacity vs demand analysis
- **What-If Calculator** — adjust patient volume, target hours, ASD %, group size, group %, and no-show rate to instantly see feasibility

## How to Use

Open `index.html` in any modern browser. No build step, no dependencies, no server required.

## Context

Built for a program seeing **200 new patients/year** (400 active) with a 7.8 FTE team (NPs, dietitians, kinesiologist, exercise physiologist, social worker, psychologist). Approximately 15% of patients (ASD cohort) remain on a fully individualized 1:1 track.
